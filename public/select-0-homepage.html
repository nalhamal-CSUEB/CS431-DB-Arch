Display Table Contents

Select a table from the left panel of options. The contents of the entire table will be displayed.
The table headings have arrow buttons that will resend the query and change the order the data is displayed (order by).

There is an update feature. On the right hand side of any table, there is a column that is currently empty. If you click there, you will get an option to update the table or cancel. The buttons "Update" and "Cancel" are visible at that time.

The last feature I was able to implement is an insert button. Unfortunately, this isn't very dynamic, as it doesn't give the User options to select from a drop down menu, or to leave values empty and allow for autoincrement to be completed by the database.

This project was completed using mysql Workbench and mysql database. NodeJS and packages Express, mysql2, and path.
